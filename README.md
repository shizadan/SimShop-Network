# SimShop Network — Business Management System

A production web application built for Shizaye Multilinks Limited, 
a multi-branch telecoms retail business in Kaduna, Nigeria.

## Live App
https://simshop.netlify.app

## What It Does
SimShop Network is a real-time business management system that helps 
a multi-branch telecoms retail chain track sales, stock, revenue, 
and distribution across all locations.

## Features
- 🔐 Multi-role authentication — Admin, CEO, Branch Manager
- 📊 Real-time dashboards per role
- 💰 Transaction recording — Sales, Swaps, Free SIMs, Failed
- 📦 Stock management with low stock alerts
- 🚚 Distribution tracking — pickups, manager & retailer deliveries
- 📈 Analytics — weekly, monthly, annual breakdowns
- 👥 Customer database per branch
- 📋 Custom date range reports

## Tech Stack
- HTML, CSS, JavaScript (Vanilla)
- Firebase Authentication
- Cloud Firestore (real-time database)
- Hosted on Netlify

## Architecture
Three distinct user roles with different access levels:
- **Admin** — Full access, manage shops and users
- **CEO** — Read-only network overview and analytics
- **Manager** — Branch-level recording and reporting
